# 🐕 Hunde Trainings App

## Projekt-Infos

- Expo / React Native App (TypeScript)
- Supabase Backend: https://axkkhyqrjrtbkumaulta.supabase.co
- Sprache: Deutsch, per Du
- Design: Dark UI, minimalistisch, Premium Look

## Commands

- `npx expo start` — Dev Server starten
- `npx expo start --ios` — iOS Simulator
- `npx expo start --android` — Android Emulator

## Architektur

- /app — Expo Router Screens
- /components — Wiederverwendbare Komponenten
- /screens — Screen-Komponenten
- /services — API/Supabase Services
- /lib — supabase.ts, helpers
- /hooks — Custom Hooks
- /store — Zustand Stores
- /types — TypeScript Types
- /assets — Bilder, Fonts

## Tech Stack

- expo-router für Navigation
- @supabase/supabase-js für Backend
- zustand für State Management
- react-native-calendars für Kalender
- TypeScript strict mode

## Design-Vorgaben

- Dunkles UI (#0A0A0F Background)
- Accent: #6C63FF (Lila)
- Karten-basiertes Layout
- Moderne Animationen
- Holländischer Schäferhund als Branding

## Coding-Konventionen

- TypeScript bevorzugen, kein `any`
- Modulare Komponenten
- Expo-kompatibel
- Mobile-first
- Sauberer Produktionscode
