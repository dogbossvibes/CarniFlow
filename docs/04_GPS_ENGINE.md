# GPS ENGINE

## Zweck

Die GPS Engine ist ausschließlich für Positionen verantwortlich.

Sie kennt keine Gegenstände.

Sie kennt keine Winkel.

Sie kennt keine Teilstrecken.

---

# Aufgaben

Die GPS Engine:

- liefert Positionen

- filtert Ausreißer

- liefert Heading

- liefert Geschwindigkeit

- speichert TrackPoints

Mehr nicht.

---

# Nicht Aufgabe

Die GPS Engine entscheidet NICHT:

- wann ein Gegenstand gesetzt wird

- wann ein Winkel gesetzt wird

- wann eine Teilstrecke beginnt

Diese Entscheidungen trifft ausschließlich das Fährtenmodul.
